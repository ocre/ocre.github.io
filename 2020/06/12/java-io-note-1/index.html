IO流的概念和分类

什么是流

读写数据时像流水一样，从一端到另一端，因此叫做“流”。

流的分类

按处理内容分为：

  • 字节流
  • 字符流。 （双字节）

按输入输出类别分为：

  • 输入流
  • 输出流

按跟数据源的关系分为:

  • 节点流
  • 处理流
